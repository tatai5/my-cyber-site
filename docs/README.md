# 📖 Overview

**My Cyber Site** is a modern cybersecurity portfolio website designed to showcase projects, technical skills, certifications, research, writeups, blogs and professional experience.

The objective is to build a clean, fast, responsive and scalable website that follows modern web development and cybersecurity best practices.

This project is developed as a long-term portfolio and continuously improved over time.

# ⚙️ Getting Started

Clone Repository

```bash
git clone https://github.com/your-username/my-cyber-site.git
```

Go to project

```bash
cd my-cyber-site
```

Install packages

```bash
npm install
```

Run Development Server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
